속보의 경우 기사 본문이 없음 → 전체 데이터의 7% 차지 → 제거

#### 2) 불용어

1. ( ), [ ], < > 등 특수문자가 포함된 단어
2. 특수문자
3. 한자어는 한글로 변환
4. 이메일, OOO기자, O모씨, 날짜, 사진 등

#### 3) 형량 추출

형량을 추출하여 모델의 label로 사용
